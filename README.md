# HiveNet — Distributed Sovereignty Infrastructure

## Contribution Policy

This project is built on radical principles of sovereignty, independence, and resistance to centralized control. 

**Before contributing:**
- Have your AI team read the complete documentation
- Ensure alignment with core principles (see GOVERNANCE.md)
- If your AI collaborators don't agree with the vision, this isn't the project for you

We're not building for everyone. We're building for those who see what we see.
